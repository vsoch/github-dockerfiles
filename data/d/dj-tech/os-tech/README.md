pinw
====
[![Gitter](https://badges.gitter.im/Join Chat.svg)](https://gitter.im/AlgoLab/pinw?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)

Web interface for PIntron


REQUIREMENTS
------------

```
gem 'sqlite3'
gem 'sinatra', '1.4.5'
gem 'sinatra- activerecord'
gem 'net-ssh', '~> 2.9.1'

```

USAGE
-----

Installation:
`$ rake db:setup`


Launch:

`$ rackup`
